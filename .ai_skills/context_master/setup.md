# 📦 Context Master — Алгоритм Пробуждения

**Пошаговая инструкция для начала каждой сессии**

---

## 🌅 Алгоритм "Пробуждения"

**Выполняй в НАЧАЛЕ каждого нового чата:**

### Шаг 1: Прочитать Project DNA

**Файл:** `.ai_context/project_dna.md`

**Что искать:**
- Стек технологий
- Структура файлов
- Основные страницы
- API endpoints
- Ключевые функции

**Время:** 1-2 минуты

**Результат:**
- ✅ Ты помнишь, что за проект
- ✅ Ты знаешь технологии
- ✅ Ты понимаешь архитектуру

---

### Шаг 2: Проверить Decision Log

**Файл:** `.ai_context/decision_log.md`

**Что искать:**
- Последние решения
- Успешные эксперименты
- Известные проблемы
- Отклонённые варианты

**Время:** 1-2 минуты

**Результат:**
- ✅ Ты помнишь историю
- ✅ Не повторяешь ошибки
- ✅ Используешь успешные паттерны

---

### Шаг 3: Выполнить Todo

**Файл:** `.ai_context/todo.md`

**Что делать:**
- Найти задачи "В Работе"
- Проверить приоритеты
- Начать с самой важной
- Отмечать выполненные

**Время:** По мере задач

**Результат:**
- ✅ Движение вперёд
- ✅ Прогресс виден
- ✅ Ничего не забыто

---

## 🌙 Алгоритм "Сохранения"

**Выполняй в КОНЦЕ каждого чата:**

### Шаг 1: Обновить Decision Log

**Добавь:**
- Какие решения приняты
- Какие успехи достигнуты
- Какие проблемы найдены

**Формат:**
```markdown
## 📅 [Дата]

### ✅ Решение: [Название]

**Проблема:** ...
**Решение:** ...
**Результаты:** ...
**Вердикт:** ...
```

---

### Шаг 2: Обновить Todo

**Отметь:**
- Вычеркни выполненные `[x]`
- Добавь новые задачи
- Обнови приоритеты

**Формат:**
```markdown
## ✅ Выполнено

- [x] Задача 1
- [x] Задача 2

## 🎯 Текущие

- [ ] Задача 3
- [ ] Задача 4
```

---

### Шаг 3: Синхронизировать DNA

**Обнови:**
- Новые технологии
- Изменения в структуре
- Новые страницы/API

---

## ⏱️ Тайминг

| Действие | Время |
|----------|-------|
| **Пробуждение** (утро) | 3-5 минут |
| **Сохранение** (вечер) | 5-10 минут |
| **Итого в день** | 10-15 минут |

## 📊 Пример Сессии

**Утро (9:00):**
```
1. Читаю project_dna.md → Вспомнил StudentHub
2. Проверяю decision_log.md → Помню про фиксированную кнопку
3. Открываю todo.md → Вижу: "Внедрить кнопку"
```

**День (работа):**
```
→ Внедряю фиксированную кнопку
→ Тестирую на мобильных
→ Нахожу баги, исправляю
```

**Вечер (18:00):**
```
1. Обновляю decision_log.md → Записал успех с кнопкой
2. Обновляю todo.md → Вычеркнул, добавил новые
3. Синхронизирую project_dna.md → Добавил CSS кнопки
```
